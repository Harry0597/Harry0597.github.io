Check in for RSA

Checkin

题目

题解

法1

  • n中有大合数（r），可以把r舍掉

    即： n = n // r

  • 对于n = p*q, 求d需要知道其phi值，观察发现，$hint1 = p(p-1) (q-1)$

    求d的时候，也可用整数倍的phi，即：直接将hint1当phi

法2

  • GCD(n, hint1)后有含p的数
  • 求r的phi
  • e、phi不互素
